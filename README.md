# Programare Funcțională - FMI

Acest repository conține laboratoarele realizate pentru cursul de **Programare Funcțională** din cadrul Facultății de Matematică și Informatică (FMI). 

## Descriere

Repository-ul include fișiere `.hs` cu exerciții practice și laboratoare care acoperă concepte fundamentale de programare funcțională, cum ar fi:

- Funcții pure și recursie
- Liste și operații pe liste
- Funcții de ordin superior
- Pattern matching
- Manipularea datelor imutabile

## Structura proiectului

- `lab1.hs` – Exerciții introductive și bazele limbajului Haskell
- `lab2.hs` – Funcții și recursie
- `lab3.hs` – Liste și operații pe liste
- `lab4.hs` – Funcții de ordin superior
- `lab5.hs` – Pattern matching și expresii condiționale
- `lab6.hs` – Proiect final sau exerciții avansate
